# Basim public-safety / urgent-event carousel notes

Use this reference when Basim asks for a carousel tied to a fresh, urgent event that should help many people (earthquake, tsunami warning, flood, severe weather, public-safety alerts).

## Source and angle

- Verify against an official or near-official source first, especially BMKG/BPBD/BNPB/government or reputable media quoting them.
- Put the concrete update early, but avoid alarmism. Example angle from the June 2026 BMKG case: **"peringatan sudah berakhir, tapi checklist keselamatan belum."**
- If an alert has ended, do not imply the danger is fully gone. Shift the usefulness toward post-event safety checks, aftershocks, building inspection, family plans, and official-source monitoring.
- Always include that warning/status can change quickly and people should follow official channels.

## Carousel shape that worked

For @ceritaberkelas-inspired Basim safety carousels:

- Format: portrait 4:5, 1080×1350.
- Mood: calm, pastel, minimal, reflective, saveable — not dark, techy, sensational, or emergency-poster noisy.
- Structure: 6–8 slides, one clear thought per slide.
- Copy pattern:
  1. Urgent but calm hook.
  2. Common wrong assumption.
  3. 60-second practical checklist.
  4. Location/risk-specific reminder.
  5. Emergency kit / family action.
  6. Simple household rule.
  7. Save/share + official-source caveat.
- Use phrases like "tenang, tapi tetap waspada" and "simpan/kirim ke orang rumah" when useful.

## Production workflow

- Generate a contact sheet/preview plus individual slide PNGs and a ZIP when the user asks for carousel assets.
- For text-heavy carousels, deterministic rendering (HTML/SVG/Pillow) is often safer than pure image generation because spelling, line breaks, and official facts must stay accurate.
- If the user explicitly names a model such as "GPT Image 2" but the configured image tool does not expose model selection, still produce the assets with the available toolchain and say briefly that the deliverable is ready; do not invent a model-specific claim.
- Verify dimensions and file type before final delivery.
